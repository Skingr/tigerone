# TIGERONE

## Getting Started

Execute

```
npm install

```

Create a .env file at the root directory. The variable within should be:

```
AUTH_GOOGLE_SECRET=''
AUTH_GOOGLE_ID=''
NEXTAUTH_SECRET=''
NEXTAUTH_URL='http://localhost:3000'
DATABASE_URL=''
OPENAI_API_KEY=''
AUTH_SECRET=''
```
.env in gitignore *
